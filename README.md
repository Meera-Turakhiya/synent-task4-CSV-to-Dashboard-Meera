# Iris Dataset - Interactive Dashboard Development

## Overview

Dashboards provide an effective way to visualize and monitor data through interactive charts and filters. This project focuses on developing an interactive dashboard using the Iris dataset to explore patterns, compare species, and gain insights through multiple visualizations.

## Objective

The objective of this project is to create an interactive dashboard that visualizes patterns within the Iris dataset and allows users to explore data dynamically.

## Dataset

The Iris dataset contains measurements of iris flowers, including Sepal Length, Sepal Width, Petal Length, Petal Width, and Species information.

## Tools and Technologies

* Power BI
* Microsoft Excel / CSV Dataset
* Data Visualization Techniques

## Tasks Performed

### 1. Data Preparation

* Imported the Iris dataset into Power BI.
* Verified data structure and field consistency.
* Prepared the dataset for dashboard creation.

### 2. Dashboard Design

* Created a dashboard layout for effective visualization.
* Organized multiple charts and filters into a single interface.

### 3. Visualization Development

* Developed a species count chart.
* Created a scatter plot to compare Sepal Length and Petal Length.
* Built a histogram for Sepal Length distribution.
* Added a heatmap to compare Petal Length values across species.

### 4. Interactive Features

* Implemented a Species Filter for dynamic analysis.
* Enabled interactive exploration of individual flower species.

### 5. Insight Generation

* Analyzed species distribution.
* Compared feature measurements across species.
* Identified distinguishing characteristics among flower categories.

## Results

The dashboard successfully combined multiple visualizations into a single interactive platform. Users can efficiently explore the Iris dataset, compare species, and uncover meaningful patterns through interactive filtering.
